# Objective

Build a **full backend for a MERN stack web application** that manages **role applications for projects, communities, startups, and organizations**.

The system should allow:

### Applicants

- Apply for roles
- Upload CV
- Write motivational letter
- Provide profile details
- Track application status

### Admins

- View all applications
- Filter by role
- See application time
- Review CVs
- Approve / reject candidates
- Add notes
- Export candidate lists

The system must also support **future hiring use cases** where organizations can use the platform to **find talent and manage recruitment pipelines**.

# Tech Stack

Backend:

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Multer (file uploads)
- Cloud storage support (Cloudinary / S3 optional)

Security:

- Helmet
- Rate limiting
- Input validation
- JWT auth middleware
